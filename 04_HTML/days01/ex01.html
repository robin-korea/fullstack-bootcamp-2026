HTML 수업 1일차

  • HTML
    1. HyperText Markup Language
    2. 웹 페이지의 구조(뼈대)를 만드는 마크업 언어
    3. 태그(Tag) 를 이용해서 문서의 구조를 정의: 시작태그, 닫기태그
    4. Element(요소) = 시작태그 + 내용(Content) + 닫기태그

  • HTML 문서의 기본 구조
    1. <!DOCTYPE html> 문서 유형 선언문, HTML5 문법을 따르는 문서, 대문자 권장
    2. HTML : HTML 문서의 시작 ~ 끝, 루트(root) 엘리먼트, 모든 요소의 최상위 부모 요소
    3. HTML > head : HTML 문서의 메타 정보가 포함. 필수 X
    4. HTML > head > meta : 인코딩 UTF-8 메타 정보
    5. HTML > head > meta : 브라우저의 제목 지정
    6. HTML > body : 필수 O, 문서의 본문(제목, 이미지, 표, 목록 등등) 을 정의하고 보여지는 모든 컨텐츠를 담는 요소(컨테이너)
